# SuperK (SuperPowers + Andrej Karpathy Skills)

This repository is a combination of two amazing AI agent skills:
- **Superpowers** (from `obra/superpowers`)
- **Andrej Karpathy Skills** (from `multica-ai/andrej-karpathy-skills`)

I really love these skills, which is why I decided to combine them into one unified toolkit. Enjoy!


##  🧩 Skills Overview
- **superpowers**: A powerful collection of agent skills for automation and productivity. ([superpowers](https://github.com/obra/superpowers))
- **andrej-karpathy-skills**: A comprehensive suite of agent skills curated by Andrej Karpathy. ([andrej-karpathy-skills](https://github.com/multica-ai/andrej-karpathy-skills))

## 🚀 Installation

You can install this skill toolkit directly into your project using `npx` (via degit):

```bash
npx degit widisaadi/superk-skills ./skills
```

## 💡 How to Use

Using SuperK is incredibly simple. Just mention **superk** in your prompt!

For example:
> "Agent, please use **superk** to debug this error."

The agent will automatically utilize the combined superpowers and Karpathy guidelines.
